## Sistem Peminjaman Barang

Aplikasi Sistem Peminjaman Barang ini bertujuan untuk mempermudah proses
pengelolaan data peminjaman barang di sebuah institusi atau organisasi. Sistem ini
memungkinkan pengguna untuk melakukan peminjaman, pengembalian, serta
pengelolaan stok barang secara efisien dan terdata dengan baik.
